# Sample-NodeJS-OpenAI
## To run this project, please following the below instructions
* Go to the website OpenAI `https://openai.com/` to create your own account.
* Generate a secret key for using the api with your above account
* Create the file .env in this directory and add the content. Please replace with your OpenAI key.
`OPENAI_API_KEY=<Your-open-ai-key>`
* Run the command to install all dependencies: `npm install`
* Run the command run this application: `npm start`
* Open browsers and type this URL `http://localhost:5001?language=<your-language>`. Please replace your language in this URL.
## Happy coding
